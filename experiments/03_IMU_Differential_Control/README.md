1. 实验背景
在医疗拖行机器人的实际应用中，由于病床属于重载设备（含患者及器械），在拖行过程中受惯性、地面摩擦力不均或机械结构间隙影响，极易产生航向角偏移（Yaw Drift） 。这种偏移在进入窄路或电梯时可能导致碰撞事故。本项目旨在利用高精度传感器实现底盘的实时复位纠偏。

2. 核心硬件：工业级高精度 IMU
区别于消费级传感器（如 MPU6050），本实验采用了战术级/工业级高精度陀螺仪：

核心优势：极低的零偏不稳定性（Bias Instability）和超低噪声密度。


应用价值：在重载大惯性场景下，能够捕捉到 0.1 度以内的微小偏航，为精密控制提供准确的数据源 。

3. 技术实现
本实验核心代码 imu_diff_correction.c 展示了从数据感知到动力执行的闭环控制过程：


数据化处理：对 IMU 输出的高频原始数据进行滤波处理，通过 ROS/Linux 话题发布订阅机制进行实时同步 。

PID 纠偏算法：以目标航向角为基准，将实时偏航角作为反馈量，通过 PID 闭环算法 计算出补偿值。

差速控制模型：根据补偿值动态调整左右驱动轮的速度差（Differential Speed），实现机器人在行进过程中的“无感”复位。

4. 实验成果

高精度复位：成功解决了重载病床在大惯性下的轨迹漂移问题 。


稳定性提升：在医院窄路行驶和进出电梯场景下，通过高精度的角度补偿，使航向误差始终控制在设定阈值以内，确保了医疗运输的安全性和平稳性 。

1. 实验背景
在智慧医疗场景中，护理机器人需要确保药品准确送达对应患者手中。本项目开发了一套基于人脸识别的身份验证系统，代替传统的刷卡或人工确认，提升了医院药事管理的自动化与安全性。

2. 技术栈
视觉处理：使用 Python + OpenCV 进行人脸特征提取与实时比对。

系统环境：运行于 Linux (Ubuntu) 上位机，通过远程操控进行调试。

跨平台通信：识别结果通过串口或自定义协议同步至 STM32 底盘，触发机械结构动作。

3. 核心功能与逻辑
身份比对：通过摄像头实时采集患者面部信息，并与预存数据库进行特征值匹配。

自动化取药：当人脸比对置信度超过设定阈值时，系统自动下发“开启药仓”指令，实现无接触取药。

预处理优化：针对医院复杂的光影环境，加入了图像预处理与归一化逻辑，有效提升了识别率的稳定性。

4. 实验亮点
全链路闭环：实现了从“视觉识别”到“逻辑判断”再到“硬件执行”的完整流程。

高安全性：通过算法层面过滤非目标人员，确保了医疗场景下的身份唯一性校验。
